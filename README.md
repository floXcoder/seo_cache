# SeoCache

Cache dedicated for SEO with Javascript rendering :fire:


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'seo_cache'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install seo_cache
